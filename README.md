# Block Combo Puzzle Prototype

8×8の盤面に3つのブロックを配置して、縦横ラインを消すブラウザゲームのプロトタイプです。

## 遊び方

1. 右側 / 下側の3つのブロックから1つ選ぶ
2. 盤面の置きたい場所をクリック / タップ
3. 縦または横が埋まるとラインが消える
4. 置けるブロックがなくなるとゲームオーバー

## 構成

- `index.html`：画面構造
- `style.css`：見た目
- `script.js`：ゲームロジック

## ローカルで開く方法

`index.html`をブラウザで開くだけで動きます。
